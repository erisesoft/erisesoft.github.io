eRisemail 电子邮件系统

下载安装 | 获取支持 | 问题反馈
  • 基于 Linux 的分布式可扩展的邮件服务器和即时消息服务器。
  • 支持 SMTP, POP3, IMAP4, HTTP, XMPP, LDAP 和 SSL/TLS。
  • 支持部署系统至 Hadoop HDFS 上。
  • 支持 WebMail 和 WebAdmin。
  • 支持 LDAP 地址簿。
  • 支持 CRAM-MD5, DIGEST-MD5, APOP, EXTERNAL/TLS-Certification 和 GSSAPI/Kerberos。
  • 支持邮件组，自定义策略和外发邮件审核。
  • 支持当前流行的邮件客户端和 IM 客户端。
  • 支持日历系统（基于 Mozilla Thunderbird Lightning Plugin）
  • 支持大邮件（最大2G）。
  • 支持 Apache SpamAssassin 以及可自定义 反垃圾邮件引擎。
