eRisemail 电子邮件系统

下载 | 文档 | 支持 | 版权

系统介绍

  • 基于 Linux 的分布式可扩展的 E-mail 服务器和 IM(基于XMPP协议)服务器。
  • 支持 SMTP, POP3, IMAP4, HTTP, XMPP, LDAP 和 SSL/TLS。
  • 支持部署系统至 Hadoop HDFS 上。
  • 内置 Web Mail 和 Web Admin 功能。
  • 支持 LDAP 地址簿。
  • 支持 CRAM-MD5, DIGEST-MD5, APOP, EXTERNAL/TLS-Certification 和 GSSAPI/Kerberos。
  • 支持邮件组，自定义策略和外发邮件审核。
  • 支持当前流行的邮件客户端和 IM 客户端(例如 Pidgin)。
  • 支持基于 Mozilla Thunderbird Lightning Plugin 的日历系统。
  • 支持最大 2G 的大邮件。
  • 支持 反垃圾邮件引擎扩展 和第三方引擎，如 Apache SpamAssassin。
